# ヒーロー背景画像の置き場

トップページの背景に写真を使う場合、このフォルダに以下のファイル名で画像を置いてください
(3枚が約8秒ずつクロスフェード表示されます。無い間は自動で従来の生成風景になります)。

| ファイル名 | 表示順 |
| --- | --- |
| `hero-1.jpg` | 1枚目 |
| `hero-2.jpg` | 2枚目 |
| `hero-3.jpg` | 3枚目 |

- 推奨サイズ: 横1600px以上・1枚300KB以下目安(JPEG)
- ファイル名は小文字・拡張子 `.jpg` 固定(大文字 `.JPG` は不可)
- 1枚だけ・2枚だけでも動作します(無いスライドは生成風景が見えます)

**注意(画像の権利):** ここに置いた画像は公開サイトに表示されます。
自分で撮影した写真、掲載許諾を得た写真、ライセンス上ウェブ掲載可能な写真のみを使用してください。
宿泊施設の公式サイト等から取得した写真の無断掲載は著作権侵害になるおそれがあります。
